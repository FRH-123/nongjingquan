# 农经权二轮延包可视化平台 — 产品需求文档 (PRD)

> **文档版本**：v1.0  
> **编制日期**：2025-07-11  
> **产品名称**：农经权二轮延包可视化平台  

---

## 一、产品概述

农经权二轮延包可视化平台是一个面向管理层的**数据监控看板系统**，接入内业数据处理软件输出的确权数据包，自动聚合统计各村组摸底调查、内业审核、合同签订、确权发证等关键进度指标，以大屏可视化方式一目了然地呈现项目整体推进情况，帮助领导快速掌握进度、发现滞后区域、精准调度决策。

---

## 二、目标用户

| 角色 | 核心诉求 | 典型场景 |
|------|----------|----------|
| 省/市级领导 | 宏观进度总览、区域横向对比 | 大屏查看全省/全市推进率排名，定位滞后区县 |
| 县级领导 | 各乡镇/村组进度明细 | 按村组下钻查看"摸底了多少户、审核通过多少户" |
| 乡镇项目负责人 | 发现本乡镇进度瓶颈 | 查看"哪些村组未上报、哪些卡在审核环节" |
| 数据管理员 | 导入数据、保障数据质量 | 上传内业软件输出的确权数据包，查看导入校验结果 |

---

## 三、核心功能清单

### P0 — 首期必须交付

| 编号 | 功能 | 说明 |
|------|------|------|
| P0-1 | 数据包导入 | 上传内业软件输出的 ZIP 数据包（含 MDB + Shapefile + XLS），自动解析入库、坐标转换、校验 |
| P0-2 | 核心指标聚合 | 导入后自动计算：发包方总数、承包方总数、已摸底户数、已内业审核数、已完成户数 |
| P0-3 | 监控大屏 — 全局概览指标卡 | 顶部横向展示 5 个核心指标卡片：发包方总数 / 承包方总数 / 已摸底户数 / 已内业审核数 / 已完成户数 |
| P0-4 | 监控大屏 — 权属单位筛选 | 左侧下拉框按行政区划（村/组）筛选，切换后全屏数据联动刷新 |
| P0-5 | 监控大屏 — 承包地块用途统计 | 左侧分类统计卡片：种植业 / 林业 / 畜牧业 / 渔业 地块数量 |
| P0-6 | 监控大屏 — 摸底填报情况统计 | 左侧分组柱状图：按村组对比"已上报"与"已审核"户数 |
| P0-7 | 监控大屏 — 天地图 | 中央区域展示行政区划地图，高亮标注村组位置，支持缩放与定位 |
| P0-8 | 监控大屏 — 摸底填报统计饼图 | 右侧环形饼图：展示分户保留户 / 新增户 / 地块转 等分类占比（分页切换，共 3 组） |
| P0-9 | 监控大屏 — 内业审核表格 | 右侧数据表格：承包方名称 / 村组 / 上报类型 / 当前状态，支持分页 |
| P0-10 | 行政区划树 | 解析权属单位代码表，构建县→镇→村→组层级树 |

### P1 — 二期增强

| 编号 | 功能 | 说明 |
|------|------|------|
| P1-1 | 大屏村组下钻 | 点击地图村组区域或柱状图村组名称，弹出该村的进度详情面板 |
| P1-2 | 确权发证进度监控 | 新增指标卡与图表：合同签订率 / 发证率 / 领证率 |
| P1-3 | 地块地图叠加 | 天地图上叠加地块多边形边界（从 Shapefile 渲染） |
| P1-4 | 进度趋势折线 | 底部区域展示按日/周的趋势变化曲线 |
| P1-5 | 数据质量看板 | 独立页面：空间重叠、面积异常、编码校验异常等数据质量指标 |
| P1-6 | 历史数据包版本对比 | 支持对比两次导入的进度差异（本周 vs 上周） |

### P2 — 三期扩展

| 编号 | 功能 | 说明 |
|------|------|------|
| P2-1 | 土地流转监控 | 流转率 / 流转面积 / 即将到期流转合同预警 |
| P2-2 | 补贴发放监控 | 补贴发放率 / 已发放金额 / 未发放清单 |
| P2-3 | 预警通知 | 进度滞后预警（完成率低于阈值）、流转到期预警、数据质量预警，站内消息推送 |
| P2-4 | 移动端适配 | 响应式布局，支持领导手机端查看 |
| P2-5 | 多级权限管控 | 按省/市/县/镇行政级别限制数据可见范围 |

## 四、页面结构

### 4.1 页面清单

| 页面 | 路由 | 功能 | 优先级 |
|------|------|------|--------|
| 数据监控大屏 | `/dashboard` | 全局监控看板，核心页面 | P0 |
| 数据导入管理 | `/import` | 上传数据包、查看导入记录与校验结果 | P0 |
| 登录页 | `/login` | 用户认证 | P0 |
| 村组详情弹窗 | `/dashboard` (弹窗) | 下钻查看单个村组的进度明细 | P1 |
| 数据质量看板 | `/quality` | 数据质量指标与异常明细 | P1 |
| 历史趋势对比 | `/trend` | 进度趋势折线与版本对比 | P1 |
| 流转监控 | `/transfer` | 流转率、到期预警 | P2 |
| 补贴监控 | `/subsidy` | 补贴发放进度 | P2 |
| 预警中心 | `/alert` | 预警事件列表与规则配置 | P2 |
